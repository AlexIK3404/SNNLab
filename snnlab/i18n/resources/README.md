# Interface localization resources

Short RU/EN strings used by the desktop interface.

Короткие RU/EN-строки интерфейса.

Keep the key sets of `en.yaml` and `ru.yaml` synchronized. Internal IDs, enum values, configuration keys, and serialized data must stay language-independent.

Наборы ключей `en.yaml` и `ru.yaml` должны совпадать. Внутренние ID, enum-значения, ключи конфигурации и сериализованные данные не должны зависеть от выбранного языка.
