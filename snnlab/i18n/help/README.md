# Study-mode help resources

Extended RU/EN explanations shown when study mode is enabled.

Расширенные RU/EN-пояснения, используемые режимом изучения.

Each topic should use the same stable key in `en.yaml` and `ru.yaml`. Keep explanations focused on what a parameter/concept means, what increasing/decreasing it changes, and important caveats.

Каждая тема должна иметь одинаковый стабильный ключ в `en.yaml` и `ru.yaml`. Пояснения должны описывать смысл параметра, влияние изменения значения и важные ограничения.
