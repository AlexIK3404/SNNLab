# Core

Shared low-level building blocks used across architectures and experiment types.

Общие низкоуровневые компоненты, используемые разными архитектурами и типами экспериментов.

## Contents

- numerical integration methods for Izhikevich neurons;
- input encoders;
- framework events/observer interfaces;
- metrics helpers;
- component registries/catalogs;
- reusable readout construction for Reservoir experiments.

This package should remain GUI-independent and as architecture-agnostic as practical.

Этот пакет не должен зависеть от GUI и по возможности должен оставаться независимым от конкретной архитектуры.
