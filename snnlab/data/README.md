# Data

Dataset loading and deterministic preparation utilities.

Утилиты загрузки датасетов и воспроизводимой подготовки данных.

The package currently provides the dataset preparation used by DCI/Reservoir experiments. Dataset selection, normalization, deterministic subset creation, and train/test pool handling belong here rather than in GUI code.

Здесь должны находиться выбор датасета, нормализация, детерминированное формирование подвыборок и работа с train/test pool, а не логика интерфейса.
