# Configs

Typed configuration objects for runs, DCI, and Reservoir experiments.

Типизированные конфигурации запусков, DCI и Reservoir-экспериментов.

Keep default values and parameter semantics here rather than scattering them across GUI widgets or runners. GUI controls should map to these configuration objects, and serialized experiment configs should be reconstructible from them.

Основные значения по умолчанию и смысл параметров должны задаваться здесь, а не дублироваться в GUI или runner-коде.
